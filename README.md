# Exscribo



## Getting started

The platform is build with DotNet 8.0 deployed to Lambda functions.

### ui/.env
VITE_COGNITO_USER_POOL_ID=<poolid>>
VITE_COGNITO_CLIENT_ID=<clientid>
VITE_COGNITO_IDENTITY_POOL_ID=<idpoolid>
VITE_API_URL=<api_url>/prod/
